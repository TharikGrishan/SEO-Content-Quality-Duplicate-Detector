# SEO Content Quality & Duplicate Detector

## Objective
This project builds a machine learning pipeline to analyze web content for SEO quality assessment and duplicate detection, covering HTML parsing, NLP feature engineering, similarity detection, and ML classification.

## Project Structure
The code is executed entirely within the Jupyter Notebook.

## Model Performance 
* **Model:** Logistic Regression
* **Overall Accuracy:** 0.7143
* **F1-Score (Low Quality):** 0.87
* **F1-Score (Medium Quality):** 0.62
